# 自測與學習路徑顧問

> 這是一個完整的 Claude Code 熟練度自測 skill，不只是“問幾個問題”，而是會對 10 個能力域做結構化評估，並給出個性化學習路徑。

## Highlights

- 兩種模式：Quick（約 2 分鐘）和 Deep（約 5 分鐘）
- 覆蓋 10 個主題：Slash Commands、Memory、Skills、Hooks、MCP、Subagents、Checkpoints、Advanced Features、Plugins、CLI
- 能輸出總等級、分主題得分、強項、短板和建議起點
- 學習路徑會按依賴順序排序，而不是簡單羅列一堆章節
- 結果後會提供後續動作，例如從第一個短板開始、深入某個弱項或重測

## 什麼時候用

| 你可以這樣說 | 它會做什麼 |
|---|---|
| “評估一下我的水平” | 做完整自測，判斷你處於 Beginner / Intermediate / Advanced 哪個階段 |
| “我該從哪裡開始學” | 判斷你當前的起點並推薦學習順序 |
| “我接下來該學什麼” | 找出短板並給出按依賴順序排列的路線 |
| “檢查我的技能” | 輸出分主題熟練度概覽 |

## 使用方式

```text
/self-assessment
```

## 它會輸出什麼

### 1. 總體結果

- 當前等級
- 總得分（Deep 模式）
- 一句簡潔總結

### 2. 分主題能力表

會展示每個主題的：

- 得分
- 掌握程度
- 當前狀態（Learn / Review / Mastered）

### 3. 個性化學習路徑

會根據你的結果生成：

- 按依賴順序排列的階段
- 每階段推薦看的教學
- 應重點關注的知識點
- 一個建議練習
- 一個完成判據

### 4. 後續動作

測完後可以繼續：

- 從第一個短板開始學
- 深挖某個主題
- 進入練習專案
- 改用另一種測評模式重測
