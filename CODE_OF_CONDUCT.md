# Contributor Code of Conduct

## 我們的承諾

我們希望把這個專案維護成一個友好、尊重、多元、可持續合作的社群。

## 我們鼓勵的行為

- 尊重他人
- 樂於協作
- 接受建設性回饋
- 歡迎新貢獻者
- 討論問題時聚焦內容而不是攻擊個人

## 不可接受的行為

- 騷擾、歧視、侮辱
- 人身攻擊
- 威脅、恐嚇
- 惡意刷屏或故意破壞討論
- 因他人舉報不當行為而報復

## 適用範圍

適用於：

- GitHub issues
- pull requests
- discussions
- 其他與本專案相關的公開或私下溝通

## 處理原則

維護者會根據情況採取合理措施，包括提醒、警告、限制參與或移除許可權。
