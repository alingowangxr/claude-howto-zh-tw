# Claude Code 概念總覽

這個檔案用於把倉庫裡最容易混淆的幾個核心概念放在一起解釋。

## 1. Slash Commands

使用者主動輸入的快捷命令，適合顯式觸發某個動作。

## 2. Memory

長期自動載入的上下文，適合放專案規則和個人偏好。

## 3. Skills

可複用、可自動觸發的能力，適合沉澱穩定工作流。

## 4. Subagents

用於複雜任務拆分和專業分工的子代理程式。

## 5. MCP

讓 Claude 連線外部工具和實時資料的協議。

## 6. Hooks

在特定事件上自動執行動作的機制。

## 7. Plugins

把 commands、skills、MCP、hooks、subagents 打包成整套方案。

## 8. Checkpoints

用於安全試錯和回退。

## 9. CLI

Claude Code 的核心使用入口，也是自動化、腳本化和 CI/CD 的關鍵介面。
