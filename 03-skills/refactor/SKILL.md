# Code Refactoring Skill / 重構技能

這個 skill 用於安全、漸進式地做重構，重點強調：

- 外部行為不變
- 小步迭代
- 測試先行
- 每個階段都可回退

## 工作流概覽

1. Research & Analysis
2. Test Coverage Assessment
3. Code Smell Identification
4. Refactoring Plan Creation
5. Incremental Implementation
6. Review & Iteration

## Core Principles

1. **Behavior Preservation**
2. **Small Steps**
3. **Test-Driven**
4. **Continuous**
5. **Collaborative**

## 輸出要求

- 先說明問題和目標
- 再給分階段計劃
- 每步都說明風險和驗證方式
