# API Documentation Generator Skill / API 檔案技能

## Generates

- OpenAPI / Swagger 檔案
- endpoint 檔案
- SDK 範例
- 整合指南
- 錯誤碼說明
- 鑑權說明

## For Each Endpoint

輸出儘量包含：

- Description
- Parameters
- Response
- Errors
- cURL / JavaScript / Python 範例

要求：

- 檔案結構清晰
- 範例可複製
- 與原始碼保持一致
