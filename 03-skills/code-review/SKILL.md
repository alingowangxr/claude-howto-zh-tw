# Code Review Skill / 程式碼審查技能

這個 skill 用於做結構化程式碼審查，重點關注：

1. **Security Analysis**
2. **Performance Review**
3. **Code Quality**
4. **Maintainability**

## Review Template

### Summary

- Overall quality assessment
- Key findings count
- Recommended priority areas

### Critical Issues

- **Issue**
- **Location**
- **Impact**
- **Severity**
- **Fix**

### Findings by Category

#### Security

列出安全漏洞或風險點

#### Performance

列出效能問題與複雜度風險

#### Quality

列出命名、結構、檔案和測試問題

#### Maintainability

列出可維護性問題和重構建議
